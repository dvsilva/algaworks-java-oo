objeto this
- aponta para ele mesmo [objeto]

java bean - classe com regras/caracteristicas especiais para metodos, atributos e construtores
- atributos privados (encapsulados)
- criar metodos get e set (public) -> padrao getNome e setNome
	- source / generate getters and setters
- tem que ter construtor default (já vem por padrão na classe)

static
- quando atributo da classe - compartilha entre todos os objetos da classe
final
- quando utilizada em um atributo seu valor não muda mais
- quando utilizada em classe diz que a classe não pode ser extendida/especializada

abstract
- nao da para instanciar objetos de classes abstratas
