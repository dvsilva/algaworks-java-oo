historia do Java
- 1991 -> dispositivos eletrônicos inteligentes
- criaram uma linguagem Oak mudou para Java em uma cafeteria
- sofreu risco de cancelamento
- 1993 ganha popularidade com a WWW
- 1995 anuncia java em uma conferência
- 2009 Sun é comprada pela Oracle
- atualmente está em 4,5 milhões de dispositivos

plataformas Java
- JSE (standart edition) 
	- Java Plataform Standard Edition
	- JDK (Java Development Kit)
- JEE (plataforma Java EE)
	- Java Plataform Enterprise Edition
	- Servlets e JSP (Java Server Pages)
	- JSF (Java Server Faces)
	- JPA (Java Persistence API)
	- EJB (Enterprise Java Beans)
- JME (java ME)
	- Java Plataform Micro Edition
- Java Card (cartões e chips), JavaFX (web e desktop) e Java TV (middleware para ser set-top box - ginga)

como o Java evolui 
- comunidade mundial
- JCP [Java Community Process]
- JSR [Java Specification Request] -> como vai funcionar a biblioteca

máquina virtual java (Java Virtual Machine)
- analogia do tradutor, discurso e dos representantes
- contrata um tradutor e traduz o discurso para traduzir para as outras linguas
- compilar codigo fonte para binarios dos diversos sistemas operacionais
- java compila com bytecode que é interpretado pelas JVMs instaladas nos sistemas operacionais

JVM faz o java ficar lento
- não, melhora o código Just In Time (JIT)
- otimiza a execução
